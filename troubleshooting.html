Morning checklist

In Case of Disturbing Unusual Activity of Experience

  • Disconnect (say "Disconnect" – "Yes") the device from the session and connect it back to the session 
  • Disconnect all the devices from the session and connect them back to the session
  • Restart all the devices and connect them back to the session (easiest way to do that is by saying “Hey, Cortana” – “Reboot” – “Yes”).

Solving problems by yourself

If a visitor can't see anything

– Press brightness "+" button repeatedly (probably it is set to minimum)
– If it didn't help, change the device to another so people won't wait.
– Go through "In case of Disturbing Unusual Activity"

 "Connecting" message showing too long

Make sure that the device is connected to internet. Get it to the Standby mode by short pressing power button. Press power button again and make a bloom gesture. Make sure that the device is connected to Wi-Fi by checking the main menu for “MS Ford” network name.

If there is no wi-fi connection, go to Settings – Network & Internet – Wi-Fi and select the MS Ford network name. Enter Microsoft2017# as password if asked.

I don't see all the devices in list on sessions page 

Put on that device and check that the Vyzn app is running. If it is not running, start it by bloom gesture and air tapping on the Vyzn app icon. Start the Vyzn app again and scan the QR code of the session. See additional information in “Connecting a Device to the Session”.

 "Applying the anchor" message takes too long or device goes into positionning mode by itself

– Get the device to the Standby mode by short pressing power button. Press power button again. Find all Vyzn windows and air tap on “Remove”. Start the Vyzn app again and scan the QR code of the session. See additional information in “Connecting a Device to the Session”.
– Make the “Spaces Cleaning” procedure on that device.

 On one of the devices app shut itself down

Relaunch the app on that device. The playback shall start from the beginning for that device.
